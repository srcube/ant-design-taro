## Install

```bash
pnpm add @srcube/ant-design-taro
```

## Sample App

```bash
pnpm preinstall

moon atd-sample:dev-weapp
```

## Components

> Recommend replace these by using CSS or atom CSS like `Tailwind CSS` or `UnoCSS` to implement:

> - Grid
> - Space
> - AutoCenter
